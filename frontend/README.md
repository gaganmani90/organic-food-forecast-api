# Frontend

This directory contains frontend applications for the Organic Food Web Scraper project.

## Current Frontend

### Streamlit UI (`streamlit-ui/`)

The current user interface built with Streamlit.

**To run:**
```bash
cd streamlit-ui
streamlit run search_ui.py
```

**Access:** http://localhost:8501

## Future Frontend

This structure is prepared for future React-based frontend:

```
frontend/
├── streamlit-ui/        # Current Streamlit UI
└── react-app/          # Future React application (to be added)
```

When ready to add React frontend, create `react-app/` directory here.
