# Frontend

This directory contains the frontend application for the Organic Food Web Scraper project.

## React Web Application

The modern React TypeScript web application built with Vite.

**Location:** `react-web/`

**To run:**
```bash
cd react-web
npm install
npm run dev
```

**Access:** http://localhost:5173

**Documentation:** See [react-web/README.md](./react-web/README.md) for detailed setup and usage instructions.
